# Evaluación Práctica: Landing Page Responsive — Hotel Ecológico

**Estudiante:** Jair Orlando Uribe Aveiga
**Tema asignado:** Hotel ecológico
**Nombre creado para la marca:** EcoVista Hotel

## Descripción

Landing page responsive desarrollada con HTML semántico, CSS externo (mobile-first, Flexbox, CSS Grid y media queries) y componentes de Bootstrap 5. El proyecto presenta la identidad de **EcoVista Hotel**, un hospedaje ecológico que ofrece alojamiento sostenible en contacto directo con la naturaleza, aplicando la paleta de colores asignada: verde oliva, arena y blanco.
